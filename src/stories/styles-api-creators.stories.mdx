<h2>createInsertionPoint</h2>
<p>
  Creates an insertion point that the JSS compiler can use when inserting stylesheets into the document <code>head</code>. To use
  createInsertionPoint, add either a comment or element with an ID to the head. The JSS compiler will insert stylesheets at that spot in
  your markup.
</p>
<p>To use the default insertion point, 'JSS insertion point' add this comment to your `head`.</p>

```html
<!--JSS insertion point-->
```

```javascript
createInsertionPoint(userInsertionPoint: string = 'JSS insertion point'): void
```

<h3>Parameters</h3>
<ul>
  <li>
    <strong>userInsertionPoint</strong> (<i>optional</i>) a custom string in your HTML comment, or the ID of the element where you want the
    stylesheets inserted.
  </li>
</ul>

<h3>Returns</h3>
<ul>
  <li>
    <strong>void</strong>
  </li>
</ul>

<h3>Examples</h3>

```html
<html>
  <title>My Gemini website</title>
  <!--JSS insertion point-->
</html>
<body>
  <script>
    import { createInsertionPoint } from '@opensesame/gemini';
    createInsertionPoint();
  </script>
</body>
```

<h2>createStyles</h2>
<p>
  This function does nothing. It returns what it is given, but is prevents Typescript type widening. Use `createStyles` if you are creating
  your own JSS for your application. The result is typically passed to a Classes provider.
</p>

```javascript
createStyles(styles: Styles): Styles
```

<h3>Parameters</h3>
<ul>
  <li>
    <strong>styles</strong> your Styles object
  </li>
</ul>

<h3>Returns</h3>
<ul>
  <li>
    <strong>Styles</strong> object, the same one provided as an argument.
  </li>
</ul>

<h3>Examples</h3>

```javascript
const styles1: Styles = createStyles({
  buttonMargin: {
    margin: 16
  }
});

const styles2 = (theme: Theme): Styles =>
  createStyles({
    buttonMargin: {
      margin: theme.spacing.unit * 2
    }
  });
```

<h2>createStylesheet</h2>
<p>
  Used internally to create a stylesheet from a Styles object. When you pass a Styles object to your provider, the provider calls this
  function to create the CSS stylesheet from the JSS you provided.
</p>

```javascript
createStylesheet(styles: Styles, options: StyleSheetFactoryOptions = {}): StyleSheet
```

<h3>Parameters</h3>
<ul>
  <li>
    <strong>styles</strong> is a Styles object
  </li>
  <li>
    <strong>options</strong> (<i>optional</i>) StyleSheetFactoryOptions, the JSS rendering options. Available options are at{' '}
    <a href="https://cssinjs.org/jss-api?v=v10.6.0#create-style-sheet">https://cssinjs.org/jss-api#create-style-sheet</a>.
  </li>
</ul>

<h3>Returns</h3>
<ul>
  <li>
    <strong>StyleSheet</strong>
  </li>
</ul>

<h3>Examples</h3>

```javascript
const styles: Styles = createStyles({
  buttonMargin: {
    margin: 16
  }
});

const stylesheet: StyleSheet = createStylesheet(styles);
console.log(stylesheet);
/*
StyleSheet: { ... }
*/
```
