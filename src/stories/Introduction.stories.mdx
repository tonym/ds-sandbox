<p>
  The Gemini Design System by OpenSesame is a Material Design based pattern library, JSS theming engine, and Angular component library. The
  theming engine is a set of Javascript functions that work in any browser and any framework. The component library is a collection of
  Angular components that come tested and ready to use in an Angular UI.
</p>
