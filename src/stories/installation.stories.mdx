<h1>Installation</h1>
<h2 className="subheading">NPM</h2>
<p>Gemini is available as an NPM package, and can be installed using NPM or Yarn.</p>

```shell
npm install @opensesame/gemini
```

```shell
yarn add @opensesame/gemini
```

<h2 className="subheading">Fonts</h2>
<p>The default fonts for Gemini are Poppins and Open Sans, available from Google Fonts. Use by adding a link to the `HEAD` of your HTML.</p>

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700&display=swap" />
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Poppins:200,400,600,700&display=swap" />
```

<h2 className="subheading">Icons</h2>
<p>
  If using font icons, the default icons for OpenSesame are Material Icons, available from Google Fonts. Use by adding a link to the `HEAD`
  of your HTML. If using SVG icons, skip this part.
</p>

```html
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons" />
```
