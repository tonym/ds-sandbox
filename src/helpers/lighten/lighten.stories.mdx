<Meta title="Helpers/lighten" />

<h1>lighten</h1>

Lighten a color.

<h2>API</h2>

<h3>Import</h3>

Path import

```javascript
import lighten from '@opensesame/gemini/helpers/lighten';
```

Named import (use with Angular)

```javascript
import { lighten } from '@opensesame/gemini';
```

<h3>Signature</h3>

```javascript
lighten(composedColor: string, baseCoefficient: number = 1): string
```

<h3>Parameters</h3>

<ul>
  <li>
    <strong>composedColor</strong> : A valid RGB, RGBA, HSL, HSLA, or hex color string.
  </li>
  <li>
    <strong>baseCoefficient</strong> optional, default = 1 : A number between 0 and 1 representing how much to lighten the color. Any number
    is accepted, but is clamped between 0 and 1.
  </li>
</ul>

<h3>Returns</h3>

<ul>
  <li>
    <strong>string</strong>
  </li>
</ul>

<h2>Examples</h2>

<h3>Lighten an HSL color:</h3>

```javascript
const lightenedColor = lighten('hsl(0, 0%, 0%)', 0.2);

console.log(lightenedColor); // hsl(0, 0%, 20%)
```

<h3>Lighten an RGB color:</h3>

```javascript
const lightenedColor = lighten('rgb(0, 0, 0)', 0.2);

console.log(lightenedColor); // rgb(51, 51, 51)
```

<h3>Lighten an RGBA color:</h3>

```javascript
const lightenedColor = lighten('rgba(0, 0, 0, 0.8)', 0.2);

console.log(lightenedColor); // rgba(51, 51, 51, 0.8)
```

<h3>Lighten a hex color:</h3>

```javascript
const lightenedColor = lighten('#000000', 0.2);

console.log(lightenedColor); // rgb(51, 51, 51)
```

<h3>Lighten a short hex color:</h3>

```javascript
const lightenedColor = lighten('#000', 0.2);

console.log(lightenedColor); // rgb(51, 51, 51)
```

<h3>Lighten a color using the default coefficient will return the color unchanged:</h3>

```javascript
const lightenedColor = lighten('#000');

console.log(lightenedColor); // rgb(255, 255, 255)
```
