<h1>isNumber</h1>

Given an unknown value, check to see if it is a number.

<h2>API</h2>

<h3>Import</h3>

Path import

```javascript
import isNumber from '@opensesame/gemini/helpers/isNumber';
```

Named import (use with Angular)

```javascript
import { isNumber } from '@opensesame/gemini';
```

<h3>Signature</h3>

```javascript
isNumber<T>(value: T): boolean
```

<h3>Parameters</h3>

<ul>
  <li>
    <strong>value</strong> : Any value that may or may not be a number.
  </li>
</ul>

<h3>Returns</h3>

<ul>
  <li>
    <strong>boolean</strong>
  </li>
</ul>

<h2>Examples</h2>

<h3>Check a string to see if it can be a number:</h3>

```javascript
const numberTest = isNumber('5');

console.log(numberTest); // true
```
