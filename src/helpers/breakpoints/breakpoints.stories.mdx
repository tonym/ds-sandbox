<h1>breakpoints</h1>

A suite of functions for working with breakpoints.

<h1>breakpoints.between</h1>

Generate a media query for resolutions between one breakpoint and another.

<h2>API</h2>

<h3>Import</h3>

Path import

```javascript
import breakoints from '@opensesame/gemini/helpers/breakpoints';
```

Named import (use with Angular)

```javascript
import { breakpoints } from '@opensesame/gemini';
```

<h3>Signature</h3>

```javascript
breakpoints.between(begin: Breakpoint, end: Breakpoint, withAtRule: boolean = true): string
```

<h3>Parameters</h3>

<ul>
  <li>
    <strong>begin</strong> : A bounding breakpoint.
  </li>
  <li>
    <strong>end</strong> : A bounding breakpoint.
  </li>
  <li>
    <strong>withAtRule</strong> <oblique>optional, default = true</oblique> : If true, the media query will have an @ rule, if false, the
    media query will not have an @ rule.
  </li>
</ul>

<h3>Returns</h3>

<ul>
  <li>
    <strong>string</strong>
  </li>
</ul>

<h2>Examples</h2>

<h3>Create a valid media query with an @ rule:</h3>

```javascript
const mediaQuery = breakpoints.between('sm', 'lg');

console.log(mediaQuery); // @media (min-width: 600px) and (max-width: 1441px)
```

<h3>Create a valid media query without an @ rule:</h3>

```javascript
const mediaQuery = breakpoints.between('sm', 'lg', false);

console.log(mediaQuery); // (min-width: 600px) and (max-width: 1441px)
```
